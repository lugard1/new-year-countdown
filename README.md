# new-year-countdown

## Learning Objectives
- Understand how to replicate a New Year Countdown application which can be adapted for different applications for keeping track of the number of days, hours, minutes,and seconds left to enter into the year 2026 from the very first hour(midnight-1, jan. 2026) it enhances user experience, and supports beautiful user interfaces.

## Description of the Project
For the first milestone in the process of creating New Year Countdown application, you will:
- set up a new repository and prepare it for development using best practices.
